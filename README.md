# 前端开发规范

## 前言

在前端开发过程中，会慢慢发现前端因为其灵活性而导致代码逻辑混乱，代码可读性差等等，在此过程中逐渐摸索，优化前端代码，逐渐完善标准化的过程。标准化大致包括以下几个部分：

- 代码可读性
- 代码可复用性
- 文档输出
- 代码测试
- 性能优化
